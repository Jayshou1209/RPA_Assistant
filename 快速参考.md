## 📝 GUI界面快捷操作

### Token管理
```
粘贴Token → 💾保存Token → ✓测试连接
```

```
🔍爬取数据 → 查看结果 → 📊导出Excel
```

### 派工流程
```
➕派工 → 填写表单 → 提交 → 完成
```

### 退工流程
```
➖退工 → 输入订单ID → 提交 → 完成
```

```
🔄订单转派 → 填写信息 → 提交 → 完成
```

## 🆘 常见问题

### Q: 连接失败？
✅ 检查Token → 更新Token → 测试连接

### Q: 导出Excel失败？
✅ 运行: `pip install pandas openpyxl`

### Q: 程序无响应？
✅ 检查后台进程 → 查看日志 → 重启程序

### Q: Token在哪更新？
✅ GUI顶部输入框 → 💾保存Token
