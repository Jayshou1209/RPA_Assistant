# 🎯 立即开始

## 🎨 推荐：使用GUI可视化界面

### 1️⃣ 安装依赖
```
双击运行: 安装依赖.bat
```

### 2️⃣ 启动GUI程序
```
双击运行: 启动GUI.bat
```

### 3️⃣ 在GUI中更新Token
```
在Token输入框粘贴新Token → 点击"💾 保存Token"
```

**GUI界面特点：**
- ✅ 可视化操作，无需记忆命令
- ✅ 直接在界面中更新Token
- ✅ 一键导出Excel/JSON
- ✅ 实时查看操作日志
- ✅ 对话框式派工/退工/转派

---

## 📋 传统方式（命令行）

### 1️⃣ 安装依赖
```
双击运行: 安装依赖.bat
```

### 2️⃣ 配置Token
```
编辑 config.py，更新 BEARER_TOKEN
```

```
双击运行: 启动脚本.bat → 选择命令行模式
或直接运行: python main.py
```

## 🔥 核心功能

### 数据管理
✅ **数据爬取** - 自动获取司机、车辆、排班数据  
✅ **数据导出** - 支持导出为JSON或Excel格式  
✅ **Token管理** - GUI中直接更新和保存Token  

### 调度操作
✅ **派工** - 分配订单给司机  
✅ **退工** - 取消订单分配  
✅ **转派** - 订单在司机间转移  
✅ **批量操作** - 支持批量派工和退工
